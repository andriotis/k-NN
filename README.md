# k-NN
```
USAGE
        --n : number of points
        --d : number of dimensions
        --k : number of neighbors
        --brute : bruteforce search
        --sequential : sequential search
        --parallel : parallel search
        --all : all the three together search

EXAMPLE

To search the 4 nearest neighbors of a query amongst 1000 points of dimension 5 sequentially, type the following command:

        ./main --n=1000 --d=5 --k=4 --sequential
```
